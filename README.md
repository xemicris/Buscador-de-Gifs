# Buscador de Gifs
  Aplicación que muestra entre 1 y 10 Gifs obtenidos de [Giphy](https://giphy.com/).
  
  Tecnologías y herramientas utilizadas:
  
    1. React
    2. JavaScript
    3. Vite
    4. Jest
    5. Testing Library
    6. giphy API
